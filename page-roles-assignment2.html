Web Team Roles

There is five roles.

Project manager

responsible for planning and managing all the phases, requirements, resources and dependencies of the web development cycle.

Back end developer

responsible for setting up & administering web servers, database servers and other back end technologies and processes, and scripting server side languages such as PHP, JSP, ASP.

Front end developer

responsible for creating web pages and user interfaces (UI) with client side languages such as HTML, CSS, JavaScript.

Database administrator

responsible for building and maintaining databases, scripting database queries,ensuring the integrity and security of the data.

QA/test

responsible for quality assurance and testing of all web development deliverables, creating test scenarios and test cases.
